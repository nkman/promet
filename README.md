Promet
=========
Local Traffic Monitor.

Version
----
1.0


Requirements
--------------
* Gcc
* Linux

How to use
--------------

```sh
git clone git@github.com:nkman/promet.git
cd promet

gcc -o listen listen.c
./listen <protocol>

supported protocols are TCP, UDP, ICMP.
```
